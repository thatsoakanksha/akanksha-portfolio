# Akanksha Jha — Minecraft Portfolio 🧱

A creative, Minecraft-styled portfolio built for **[Akanksha Jha](https://www.linkedin.com/in/akankshajha97/)** — Strategic Consultant at **PwC**, with a background in **Design & Marketing at ISB** and an **MBA from Brunel University London**.

🌍 **Live:** 
## ✨ Features
- 🎮 Fully **Minecraft-styled** UI (pixel fonts, blocky shadows, dirt/grass panels, HUD with hearts + XP)
- 🧍 **Voxel 3D character** built procedurally with Three.js (no asset downloads)
- 🧥 **Click a "quest block" to morph the character's outfit**
  - PwC → business blazer + briefcase
  - ISB → pink design palette
  - Brunel → graduation cap
  - Adobe → artist brush
- ⌨️ Press **Space** to cycle outfits
- ☁️ Animated pixel clouds, floating character, swaying arms
- 🎒 Skills displayed as a **Minecraft inventory hotbar**
- 🏰 Past companies/schools as glowing "Realms"
- 🌌 End-portal styled contact card

## 🛠 Stack
- Vanilla HTML / CSS / JS — no build step
- [Three.js](https://threejs.org/) via CDN + OrbitControls
- Google Fonts: *Press Start 2P* + *VT323*
- Hosted free on **GitHub Pages**

## 🚀 Local preview
Just open `index.html` in a modern browser, or run a tiny static server:
```bash
npx serve .
```

## 📁 Files
- `index.html` — markup + sections
- `styles.css` — all visuals
- `main.js` — Three.js world + character + interactions
